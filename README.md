# ⏰ Digital Alarm Clock using Streamlit

## 📌 Project Overview

Digital Alarm Clock is a simple web application built using Python and Streamlit. The application displays a live digital clock and allows users to set an alarm for a specific time. When the selected alarm time is reached, the application notifies the user with an alarm sound and visual alert.

---

## 🚀 Features

- Live Digital Clock Display
- Set Alarm Time
- Visual Alarm Notification
- Alarm Sound Playback
- User-Friendly Interface
- Web-Based Application using Streamlit

---

## 🛠️ Technologies Used

- Python
- Streamlit
- Datetime Module
- Time Module

## 🎯 How It Works

1. Open the application.
2. Select the desired alarm time.
3. Click the **Set Alarm** button.
4. The digital clock updates in real-time.
5. When the current time matches the alarm time:
   - An alert message is displayed.
   - Alarm sound is played.
   - Visual animation is triggered.
